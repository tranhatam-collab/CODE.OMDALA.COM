# OMCODE_UI_SCREEN_MAP.md

UI Screen Map for OMCODE (Việt – Anh)

Mục tiêu
- Định hình bản đồ màn hình và luồng người dùng cho apps/web và desktop, đảm bảo thống nhất UX across languages.

## 1) Tổng quan màn hình chính
- Home / Landing
- Features / Capabilities
- Providers / MCP integration overview
- Docs portal overview
- API gateway status / health
- Settings / Preferences ( bilingual toggle )

## 2) Screen categories và components (shared)
- Common components: header, footer, language switcher, navigation, breadcrumbs, modals, toasts.
- UI blocks: hero, feature cards, comparison tables, provider lists, download CTA, docs index.
- Data-driven screens: using schemas and prompts as data sources.

## 3) Luồng người dùng (User Flows)
- Visitor -> Explore features -> Compare -> Get started / Download Mac
- User with account -> Access docs -> API usage -> MCP orchestration
- Admin/Maintainer -> Settings and deployments

## 4) Tên màn hình và trạng thái dịch (i18n)
- Mô tả tình trạng dịch cho từng màn hình, đảm bảo đồng bộ giữa Việt và Anh.
- Cấu trúc file i18n cho strings, key naming conventions.

## 5) Deliverables
- Wireframe hoặc mockups cho mỗi screen (vi/ja), components map, và mapping to routes.
